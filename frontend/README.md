# Siljangnim

AI-powered creative coding assistant. Chat with an AI agent that writes and runs WebGL shaders, generative art, and interactive visualizations in real-time.

## Live Demo

**https://siljangnim.vercel.app**

## Features

- Node-based canvas workspace (React Flow)
- AI agent that generates and executes WebGL scripts
- Real-time shader editing and preview
- File uploads (images, 3D models, text)
- Mobile responsive layout
- Light / dark theme

## Tech Stack

- **Frontend**: React + Vite, Tailwind CSS, React Flow
- **AI**: Anthropic Claude API (streaming)
- **Deploy**: Vercel (Edge Functions for API proxy)

## Development

```bash
cd frontend
npm install
npm run dev
```
